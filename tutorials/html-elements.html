HTML Elements

There are a lot of html elements and knowing which one is right to use can be tricky. You can build an entire website out of <div> elements and it will look great but it will not be the best when it comes to search engine optimization or screen readers for the visually impaired. To that end I have made this page to discuss not only what certain HTML elements mean but also when they should be used.

The basic elements

There are a lot of HTML elements and most develoeprs never commit them all to memory. This group of basic elements are the ones used in almost every website and will be the ones you encounter most frequently as a developer.

<h1> and other header elements

The <h1>-<h6> elements are used to title sections of your page. It typically changes the font size of the enclosed text to indicte that it is a header for what follows on the page. They have a decreasing order of importance as the number in the element tag increases. For example on this page "HTML Elements" at the top is the main heading. It tells you what to expect from the content for the entire page and so it uses an <h1> element. Further down the page you see "The basic elements" is a new section but since it falls within the HTML Elements section it has a lower heading rank so it uses an <h2> element. Finally the begining of this section on the header elements uses an <h3> element because it falls within the section titled the basic elements
